
  # TreinoGo - Professor

  This is a code bundle for TreinoGo - Professor. The original project is available at https://www.figma.com/design/mDWFQnIAhB73zw0acU0JI9/TreinoGo---Professor.

## Environment Variables

Create a `.env` file in the root directory with the following variables:

```env
VITE_API_BASE_URL=http://localhost:3001/api
VITE_LANDING_PAGE_URL=http://localhost:5173
```

Or copy from the example file:
```bash
cp .env.example .env
```

## Running the code

1. Install dependencies:
   ```bash
   npm i
   ```

2. Configure environment variables (see above)

3. Start the development server:
   ```bash
   npm run dev
   ```
